# 1. 디자인 철학

## 1.1 컨셉: "디지털 사주명리학"

Companion Foundry는 기업의 AI Agent를 마치 **운명의 동반자**를 찾는 것처럼 표현합니다. 사주팔자와 궁합의 메타포를 활용하여, AI Agent와 기업의 **완벽한 매칭**을 시각적으로 전달합니다.

| 전통 개념 | Companion Foundry 적용 |
|-----------|------------------------|
| **사주팔자** | Agent의 특성/역량 분석 |
| **궁합** | Agent와 업무의 적합도 |
| **운세** | 품질 예측 및 성능 지표 |
| **오행** | 5가지 핵심 모듈 (기획, 개발, 품질, 배포, 운영) |
| **12지신** | 12가지 산업별 Agent 템플릿 |

## 1.2 핵심 디자인 원칙

```
┌─────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────┤

```

**애니메이션 시퀀스 (총 5초):**

1. **0-1s**: 화면 중앙에서 빛의 점이 나타남
2. **1-2s**: 빛이 무지개 색으로 회전하며 폭발
3. **2-3s**: 별가루 파티클이 사방으로 퍼짐
4. **3-4s**: Agent 카드가 중앙에서 떠오르며 등장
5. **4-5s**: 축하 메시지 + 무지개 리본 애니메이션

# 9. 접근성 (Accessibility)

## 9.1 색상 대비

모든 텍스트는 WCAG 2.1 AA 기준 충족:
- 일반 텍스트: 최소 4.5:1 대비
- 큰 텍스트: 최소 3:1 대비

## 9.2 모션 접근성

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 9.3 색맹 지원

- 색상만으로 정보 전달하지 않음
- 아이콘, 패턴, 텍스트 레이블 병행 사용
- 대비 모드 지원
