# 1. 서비스 개요

## 1.1 서비스 정의

**Companion Foundry**는 기업의 복잡한 업무를 End-to-End로 자동화하는 **버티컬 AI Agent 솔루션 플랫폼**입니다. 자체 개발한 Q-Process 4단계 시스템을 통해 고품질의 AI 동반자(Companion Agent)를 빠르고 효율적으로 개발·배포합니다.

## 1.2 핵심 가치 제안

| 항목 | 내용 |
|------|------|
| **슬로건** | "기업 업무를 끝까지 함께하는 AI 동반자를 만듭니다" |
| **핵심 철학** | 벽돌(기능)이 아닌 건축물(효용)을 만든다 |
| **차별점** | 3배 빠른 개발, 70% 비용 절감, 할루시네이션 제로 도전 |

## 1.3 대상 시장 및 고객

| 세그먼트 | 타겟 고객 | 주요 니즈 |
|----------|----------|----------|
| **금융** | 증권사, 은행, 보험사 | API 자동화, 컴플라이언스 준수, 정확한 정보 제공 |
| **통신** | 통신사, 미디어 | 대규모 사용자 대응, 개인화 추천 |
| **교육** | 출판사, 에듀테크 | 맞춤 학습, 지식맵 기반 콘텐츠 |
| **공공/대기업** | 정부기관, 대기업 | 업무 자동화, 보안 준수 |

# 3. 핵심 기술 스택

## 3.1 Q-Process 4단계 시스템

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  1. 기획    │ → │  2. 개발    │ → │  3. 품질    │ → │  4. 배포    │
│ AI 기획검증 │    │ LangBridge  │    │   Veris     │    │ Agent CI/CD │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

| 단계 | 도구 | 역할 |
|------|------|------|
| **1. 기획** | AI 기획 검증 도구 | 데이터 적합성과 예상 성능 사전 검증 |
| **2. 개발** | LangBridge | 모듈형 Workflow 엔진 (개발 속도 3배↑) |
| **3. 품질** | Veris | 딥러닝 기반 Agent QA (할루시네이션 차단) |
| **4. 배포** | Agent CI/CD | 지속적 통합/배포 자동화 |

## 3.2 LangBridge (개발 엔진)

| 기능 | 설명 | 효과 |
|------|------|------|
| 모듈형 에이전트 빌더 | 검증된 블록 조합으로 빠른 구축 | 개발 속도 3배 향상 |
| 토큰 최적화 | 맥락 압축 및 캐싱 기술 | 운영 비용 70% 절감 |
| LLM 유연성 | GPT, Claude, Gemini 등 자유롭게 교체/조합 | 벤더 종속 방지 |
| 모듈형 설계 | 재사용 가능한 컴포넌트 라이브러리 | 빠른 커스터마이징 |

## 3.3 Veris (품질 관리 - Killer Feature)

| 검증 지표 | 설명 |
|----------|------|
| 정확성 (Accuracy) | 답변 내용의 사실 여부 검증 |
| 관련성 (Relevance) | 질문과 답변의 맥락 일치도 |
| 완전성 (Completeness) | 필요한 정보의 누락 여부 |
| 유해성 (Harmfulness) | 부적절하거나 위험한 내용 탐지 |
| 규정 준수 (Compliance) | 기업 내부 규정 및 법규 준수 여부 |
