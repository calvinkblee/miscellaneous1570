# 🏛️ GovGrant AI - 정부/지자체 지원사업 자동화 플랫폼

**Version: 1.0**  
**작성일: 2025. 12. 11.**  
**작성자: LaivData 기획팀**

---

## 📋 Executive Summary

### 서비스 개요
> **"정부 지원사업 공고 수집부터 지원서 작성까지, AI가 End-to-End로 동반하는 자동화 플랫폼"**

매일 50여개의 정부/지자체 지원 관련 사이트를 자동 크롤링하여 공고를 수집하고, 당사(고객사)의 사업/기술 프로필에 맞는 지원사업을 AI가 분석하여 **적합도 평가 → 지원 전략 수립 → 지원서 초안 작성 → 알림 배달**까지 원스톱으로 제공하는 B2B SaaS 서비스입니다.

### 핵심 가치 제안
| 기존 Pain Point | GovGrant AI 솔루션 |
|---|---|
| 50개 이상의 사이트를 매일 확인해야 함 | 자동 크롤링 & 통합 대시보드 |
| 공고문이 길고 복잡하여 적합성 판단이 어려움 | AI 요약 & 자동 매칭 점수 |
| 지원서 작성에 3-5일 소요 | AI 초안 생성 (30분 내) |
| 마감일 놓침, 우선순위 판단 어려움 | 스마트 알림 & 전략 추천 |

#### 🤖 Module 3: 매칭 AI (Matching Engine)

**기능**
- 기업 프로필 vs 공고 요건 자동 매칭
- 적합도 점수 산출 (0-100점)
- 지원 우선순위 추천

**매칭 알고리즘**

```
적합도 점수 = (자격요건 충족도 × 0.3) 
            + (기술 연관성 × 0.3) 
            + (성공확률 예측 × 0.2) 
            + (지원금 효율성 × 0.2)
```

**기업 프로필 구성 요소**
1. **기본정보**: 업종, 업력, 매출, 직원수
2. **기술역량**: 보유 기술, 특허, 인증
3. **사업영역**: 주요 제품/서비스, 타겟 시장
4. **지원이력**: 과거 수주 내역, 성공/실패 이력
5. **전략방향**: 단기/중기 사업 목표

**라이브데이터 프로필 예시**
```yaml
회사명: 라이브데이터
업종: AI/SW 개발
업력: 5년
매출: 50억+ (2025 예상 62억)
직원수: 32명
보유기술:
  - LLM 기반 AI Agent
  - 할루시네이션 검증 (Veris)
  - 토큰 경량화 (LangBridge)
  - 지식맵 엔진
주요고객: 한국투자증권, KT, 동아출판
타겟분야: 금융, 교육, 통신
강점키워드: [AI Agent, 생성형AI, RAG, LLM, 업무자동화]
```

#### ✍️ Module 4: 지원서 생성기 (Proposal Generator)

**기능**
- 공고 양식에 맞는 지원서 초안 자동 작성
- 기업 강점 + 공고 요건 연결 전략 수립
- 다양한 포맷 출력 (HWP, DOCX, PDF)

**생성 프로세스**

```
1. 공고 분석
   └─ 평가항목 추출 (기술성, 사업성, 성장성 등)
   
2. 전략 수립
   └─ 각 평가항목에 대응하는 기업 강점 매핑
   └─ 차별화 포인트 도출
   
3. 초안 작성
   └─ 섹션별 내용 생성
   └─ 근거자료 첨부 위치 표시
   └─ 그래프/도표 제안
   
4. 검증 & 보완
   └─ Veris로 사실 검증
   └─ 할루시네이션 체크
   └─ 용어 일관성 검토
```

**출력물 구성**

| 문서 유형 | 내용 | 분량 |
|----------|------|------|
| **전략 보고서** | 지원 이유, 성공 확률, 경쟁 분석 | 2-3페이지 |
| **지원서 초안** | 공고 양식에 맞춘 전체 내용 | 20-50페이지 |
| **체크리스트** | 필수 서류, 마감일, 담당자 할일 | 1페이지 |
| **Q&A 예상 질의** | 면접/발표 대비 예상 질문 | 10-20문항 |

## 3. 라이브데이터 기술 활용

### 3.1 핵심 기술 매핑

| 플랫폼 기능 | 라이브데이터 기술 | 활용 방식 |
|------------|------------------|----------|

| 지속적 개선 | **Agent CI/CD** | 성공 패턴 학습 및 모델 업데이트 |

### 3.2 Companion Agent 적용

**"지원사업 동반자(GovGrant Companion)"**
- 공고 발견 → 분석 → 매칭 → 지원서 작성까지 End-to-End 동반
- 사용자와 대화하며 기업 강점 파악 및 전략 수립
- 피드백 반영하여 지원서 수정/보완
